# Immer in Bewegung
Open source travel documentation app for self-hosting or serverless usage

## Live demo
You can check out a demo app running at https://go.bewegung.app/.

## Principles
* Based on JSON format
* Immer in Bewegung runs client side in your browser reading your the JSON file you bind
* If you self-host Immer in Bewegung the application can read the JSON file from the same server
* Features an optional integration to Immich (https://immich.app/)
